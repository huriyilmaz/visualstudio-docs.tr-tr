# <a name="expected--javascript"></a>Beklenen ')' (JavaScript)
Bir ifadeyi bir parantez kümesi içine almaya çalıştınız, ancak kapanış ayracı içermiyordu. Bir ifadenin açılış ve kapanış parantezleri kümesi içine alınması gerekir. Aşağıdaki örnekte ayraç kullanımına dikkat edin.  
  
```JavaScript  
for (initialize; test; increment) {  
statement;  
}  
```  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
- Değerlendirme ifadesine sağ parantezleri ekleyin.
