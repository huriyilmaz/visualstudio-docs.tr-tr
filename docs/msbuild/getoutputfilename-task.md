# <a name="getoutputfilename-task"></a>GetOutputFileName görevi

Yalnızca çıktı dizini veya tam dosya adı veya hiçbir şey belirtmeye izin cl ve diğer araçlar için çıkış dosya adı almak için yardımcı görev.

## <a name="parameters"></a>Parametreler

Aşağıdaki tabloda **GetOutputFileName** görevinin parametreleri açıklanmaktadır.

|Parametre|Açıklama|
|---------------|-----------------|
|**Çıktı Uzantısı**|Gerekli **dize** parametresi.|
|**Outputfile**|İsteğe bağlı **dize** çıkış parametresi.|
|**Outputpath**|İsteğe bağlı **dize** parametresi.|
|**Kaynak Dosya**|Gerekli **dize** parametresi.|
