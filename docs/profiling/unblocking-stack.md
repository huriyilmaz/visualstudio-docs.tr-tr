# <a name="unblock-stack"></a>Yığının engelini kaldırma
Seçili olan iş parçacığı öğesi, geçerli işlemdeki başka bir iş parçacığı tarafından engeli kaldırılmış durumdayken daha sonra yürütülmaya başlayan engellenen bir segmenti temsil ediyorsa, engellemeyi kaldıran iş parçacığı için çağrı yığını bu sekmede gösterilir.
