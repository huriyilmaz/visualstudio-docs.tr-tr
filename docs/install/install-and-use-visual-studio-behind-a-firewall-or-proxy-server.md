# <a name="install-and-use-visual-studio-and-azure-services-behind-a-firewall-or-proxy-server"></a>Visual Studio ve Azure hizmetlerini bir güvenlik duvarı veya proxy sunucusunun arkasında yükleyip kullanma

siz veya kuruluşunuz bir güvenlik duvarı veya proxy sunucusu gibi güvenlik önlemleri kullanıyorsa, Visual Studio ve Azure hizmetlerini yükleyip kullandığınızda en iyi deneyimlere sahip olmak için, açmak isteyebileceğiniz bir "allowlist" ve bağlantı noktaları ve protokoller eklemek isteyebileceğiniz etki alanı url 'leri vardır.

* **[Visual Studio yüklemek](#install-visual-studio)**: bu tablolar, istediğiniz tüm bileşenlere ve iş yüklerine erişebilmek için bir izin öğesine eklenecek etki alanı url 'lerini içerir.

* **[Visual Studio ve Azure hizmetlerini kullanın](#use-visual-studio-and-azure-services)**: bu tablo, bir izin öğesine eklenecek etki alanı url 'lerini ve istediğiniz tüm özelliklere ve hizmetlere erişebilmeniz için açılacak bağlantı noktalarını ve protokolleri içerir.

> [!NOTE]
> bu makale Windows Visual Studio için yazılmıştır, ancak bazı bilgiler ayrıca bir güvenlik duvarı veya proxy sunucusunun arkasında [Mac için Visual Studio yüklemek](/visualstudio/mac/install-behind-a-firewall-or-proxy-server) için de geçerlidir.

## <a name="troubleshoot-network-related-errors"></a>Ağ ile ilgili hataları giderme

Bazen, bir güvenlik duvarı veya ara sunucu arkasında ağ veya ara sunucu yüklemesi Visual Studio veya proxy ile ilgili hatalarda içinde çalıştırabilirsiniz. Bu tür hata iletilerine yönelik çözümler hakkında daha fazla bilgi için Ağ ile ilgili hataları yükleme veya kullanma sorunlarını [giderme Visual Studio](troubleshooting-network-related-errors-in-visual-studio.md) bakın.

## <a name="get-support"></a>Destek alın

Yüklemeyle ilgili [**sorunlar için**](https://visualstudio.microsoft.com/vs/support/#talktous) bir yükleme sohbeti (yalnızca İngilizce) destek seçeneği sunuyoruz.

İşte birkaç destek seçeneği daha:

* Ürün sorunlarını hem [IDE'de](../ide/how-to-report-a-problem-with-visual-studio.md) hem de IDE'de hem de Visual Studio Yükleyicisi Sorun Bildir Visual Studio bize bildirebilirsiniz.
* Bir özellik önerin, ürün sorunlarını izlenin ve geliştirici Visual Studio [yanıt Community.](https://aka.ms/feedback/suggest?space=8)
* Gitter [GitHub](https://github.com/) konuşmada bizimle ve diğer Visual Studio [geliştiricilerle Visual Studio hesabı kullanın.](https://gitter.im/Microsoft/VisualStudio)
