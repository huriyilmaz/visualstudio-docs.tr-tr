# <a name="install-and-use-visual-studio-and-azure-services-behind-a-firewall-or-proxy-server"></a>Güvenlik duvarı veya ara Visual Studio arkasında Azure Hizmetleri'ne ve Azure Hizmetleri'ne yükleme ve kullanma

Siz veya kuruluşta güvenlik duvarı veya ara sunucu gibi güvenlik önlemleri kullanıyorsanız, Visual Studio ve Azure Hizmetlerini yükp kullanma deneyimine sahip olmak için açmak istediğiniz bir "izin verme listesine" ve bağlantı noktalarına ve protokollere eklemek istediğiniz etki alanı URL'leri vardır.

* **[Yükleme Visual Studio:](#install-visual-studio)** Bu tablolar, istediğiniz tüm bileşenlere ve iş yüklerine erişiminiz olması için bir izin verme listesine eklemek üzere etki alanı URL'lerini içerir.

* **[Visual Studio ve Azure Hizmetlerini](#use-visual-studio-and-azure-services)** kullanma: Bu tablo, istediğiniz tüm özelliklere ve hizmetlere erişiminiz olması için bir izin verme listesine eklemek üzere etki alanı URL'lerini ve açılacak bağlantı noktalarını ve protokolleri içerir.

> [!NOTE]
> Bu makale, Visual Studio için Windows, ancak belirli bilgiler bir güvenlik duvarı veya ara sunucu [Mac için Visual Studio](/visualstudio/mac/install-behind-a-firewall-or-proxy-server) yükleme için de geçerlidir.

## <a name="troubleshoot-network-related-errors"></a>Ağ ile ilgili hataları giderme

Bazen, bir güvenlik duvarı veya ara sunucu arkasında ağ veya ara sunucu yüklemesi Visual Studio veya proxy ile ilgili hatalarda içinde çalıştırabilirsiniz. Bu tür hata iletilerine yönelik çözümler hakkında daha fazla bilgi için Ağ ile ilgili hataları yükleme veya kullanma sorunlarını [giderme Visual Studio](troubleshooting-network-related-errors-in-visual-studio.md) bakın.

## <a name="get-support"></a>Destek alın

Yüklemeyle ilgili [**sorunlar için**](https://visualstudio.microsoft.com/vs/support/#talktous) bir yükleme sohbeti (yalnızca İngilizce) destek seçeneği sunuyoruz.

İşte birkaç destek seçeneği daha:

* Ürün sorunlarını hem [IDE'de](../ide/how-to-report-a-problem-with-visual-studio.md) hem de IDE'de hem de Visual Studio Yükleyicisi Sorun Bildir Visual Studio bize bildirebilirsiniz.
* Bir özellik önerin, ürün sorunlarını izlenin ve geliştirici Visual Studio [yanıt Community.](https://aka.ms/feedback/suggest?space=8)
* Gitter [GitHub](https://github.com/) konuşmada bizimle ve diğer Visual Studio geliştiricilerle Visual Studio için hesap [hesaplarınızı kullanın.](https://gitter.im/Microsoft/VisualStudio)
