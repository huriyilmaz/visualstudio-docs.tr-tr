# <a name="ca0501"></a>CA0501

Çıkış raporu okunamıyor

Aşağıdaki nedenlerden biri nedeniyle çıkış raporu yazılamayabiliyorsa bu durum oluşur:

- Disk dolu

- Dizine yazma izni yok

- Dizin yok
