# <a name="ca0501"></a>CA0501

Çıkış raporu okunamıyor

Bu durum, aşağıdaki nedenlerden biri nedeniyle çıkış raporu yazılamaz olur:

- Disk dolu

- Dizine yazmak için izin yok

- Dizin yok
