# <a name="c28209"></a>C28209
Uyarı C28209: sembol bildiriminin çakışan bir bildirimi vardır

 Bu uyarı yanlış oluşturulmuş bir ek açıklama bildirimini gösterir. Bu uyarı, normal kullanımda asla gerçekleşmemelidir.
