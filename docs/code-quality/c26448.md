# <a name="c26448-use_gsl_finally"></a>C26448 USE_GSL_FINALLY

Son eylem amaçlanıyorsa `gsl::finally` kullanmayı düşünün.

C++Temel yönergeler: [GSL. util: Yardımcı programlar @ no__t-0

Yönergeler destek kitaplığı, *son eylem* kavramını uygulamak için uygun bir yardımcı program sağlar. C++ Dil **try-finally** yapılarını desteklemediğinden, yok etme üzerinde rastgele eylemleri çağırabilecek özel temizleme türlerini uygulamak yaygın hale geldi. @No__t-0 yardımcı programı bu şekilde uygulanır ve bir kod tabanı genelinde son eylemleri gerçekleştirmek için daha Tekdüzen bir yol sağlar.

Ayrıca, **goto** deyimlerini kullanarak (genellıkle [C26438 NO_GOTO](c26438.md)tarafından önerilmez), son eylemlerin bir eski-Fashioned C stili şekilde gerçekleştirildiği durumlar da vardır. Büyük ölçüde **goto**kullanan kodda kesin bir şekilde algılamak zordur, ancak bazı buluşsal yöntemler Temizleme için daha iyi aday bulmaya yardımcı olabilir.

## <a name="remarks"></a>Açıklamalar

- Bu kural çok hafif ve son eylem nesnelerini kullanma fırsatlarını tahmin etmek için etiket adlarını kullanır.
- Uyarı oluşturabilen etiket adları "End", "Final", "Clean" vb. gibi sözcükler içerir.
- **Goto** deyimlerinde uyarılar görüntülenir. Bazı durumlarda ayrıntılı çıkış görebilirsiniz, ancak bu durum karmaşıklığa göre kodun önceliklerini belirlemenize yardımcı olabilir.
- Bu kural her zaman [C26438 NO_GOTO](c26438.md)ile çift gider. Önceliklere bağlı olarak, bunlardan biri devre dışı bırakılabilir.

## <a name="example-cleanup-with-multiple-goto-statements"></a>Örnek: Birden çok goto deyimi ile Temizleme

```cpp
void poll(connection_info info)
{
    connection c = {};
    if (!c.open(info))
        return;

    while (c.wait())
    {
        connection::header h{};
        connection::signature s{};
        if (!c.read_header(h))
            goto end;               // C26448 and C26438
        if (!c.read_signature(s))
            goto end;               // C26448 and C26438
        // ...
    }

end:
    c.close();
}
```

## <a name="example-cleanup-with-multiple-goto-statements-replaced-by-gslfinally"></a>Örnek: GSL:: finally tarafından değiştirilmiş birden çok goto deyimi ile Temizleme

```cpp
void poll(connection_info info)
{
    connection c = {};
    if (!c.open(info))
        return;

    auto end = gsl::finally([&c] { c.close(); });
    while (c.wait())
    {
        connection::header h{};
        connection::signature s{};
        if (!c.read_header(h))
            return;
        if (!c.read_signature(s))
            return;
        // ...
    }
}
```
