# <a name="ca0055"></a>CA0055

Dosya yüklenemedi

Kod Analizi Aracı, analiz için belirtilen hedef dosyayı yükleyemedi. Dosya bulunamazsa veya dosya erişimi reddedildiyse bu durum oluşur. genellikle, bu durum kullanıcı Visual Studio giriş derleme özelliğini geçersiz kılmayı seçerse veya fxcopcmd.exe el ile çağrılabilir.
