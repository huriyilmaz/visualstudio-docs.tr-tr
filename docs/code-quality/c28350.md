# <a name="c28350"></a>C28350
Uyarı C28350: Ek açıklama \<annotation > koşullu olarak uygulanabilir olmayan bir durum tanımlar.

 Genellikle bu uyarı, C/C++ türünün incelenebildiği bir ek açıklama uygulandığında oluşturulur.
