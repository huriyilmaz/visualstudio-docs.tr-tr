# <a name="ca0056"></a>CA0056
Proje dosyası, çözümleme aracının yanlış bir sürümüne sahip.

Bu uyarı, yanlış kod Analizi Aracı sürümüne sahip bir proje dosyası ile FxCopCmd.exe el ile çağrılırsa oluşur.
