# <a name="ca0056"></a>CA0056
Proje dosyası, analiz aracının yanlış bir sürümüne sahip.

Bu uyarı, FxCopCmd.exe kod analizi aracı sürümüne sahip bir proje dosyasıyla el ile çağrılıyorsa oluşur.
