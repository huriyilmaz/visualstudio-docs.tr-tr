# <a name="c6277"></a>C6277
Uyarı C6277: @No__t-0function > çağrısında tırnak işaretsiz bir yola sahip NULL uygulama adı: yol boşluk içeriyorsa bir güvenlik açığına neden olur

 Bu uyarı, uygulama adı parametresinin null olduğunu ve yürütülebilir yol adında boşluk olabileceğini gösterir. Bu durumda, yürütülebilir dosya "tam nitelikli" olmadığı takdirde, güvenlik sorunu da olasıdır. Kötü niyetli bir Kullanıcı, yolda daha önce aynı ada sahip bir standart olmayan yürütülebilir dosya ekleyebilir. Bu uyarıyı düzeltmek için, null geçirmek yerine uygulama adını belirtebilir veya uygulama adı için null geçirirseniz, çalıştırılabilir yolun etrafında tırnak işaretleri kullanın.

## <a name="example"></a>Örnek
 Aşağıdaki örnek kod bu uyarıyı oluşturur çünkü uygulama adı parametresi null ve yürütülebilir yol adında bir boşluk vardır; işlevin alanları ayrıştırmak nedeniyle farklı bir yürütülebilir dosyanın çalıştırılabilmesi riski vardır. Daha fazla bilgi için bkz. [CreateProcess](https://docs.microsoft.com/windows/desktop/api/processthreadsapi/nf-processthreadsapi-createprocessa).

```
#include <windows.h>
#include <stdio.h>

void f_defective()
{
    STARTUPINFOA si;
    PROCESS_INFORMATION pi;

    ZeroMemory( &si, sizeof( si ) );
    si.cb = sizeof( si );
    ZeroMemory( &pi, sizeof( pi ) );
    if( !CreateProcessA( NULL,
                        "C:\\Program Files\\MyApp",
                        NULL,
                        NULL,
                        FALSE,
                        0,
                        NULL,
                        NULL,
                        &si,
                        &pi ) )
    {
        puts( "CreateProcess failed." );
        return;
    }
    // Wait until child process exits.
    WaitForSingleObject( pi.hProcess, INFINITE );

    // Close process and thread handles.
    CloseHandle( pi.hProcess );
    CloseHandle( pi.hThread );
}
```

 Bu uyarıyı düzeltmek için aşağıdaki örnekte gösterildiği gibi, yürütülebilir yol etrafında tırnak işaretleri kullanın:

```
#include <windows.h>
#include <stdio.h>

void f ()
{
    STARTUPINFOA si;
    PROCESS_INFORMATION pi;

    ZeroMemory( &si, sizeof( si ) );
    si.cb = sizeof( si );
    ZeroMemory( &pi, sizeof( pi ) );

    if( !CreateProcessA( NULL,
                        "\"C:\\Program Files\\MyApp.exe\"",
                        NULL,
                        NULL,
                        FALSE,
                        0,
                        NULL,
                        NULL,
                        &si,
                        &pi ) )
    {
        puts( "CreateProcess failed." );
        return;
    }
    // Wait until child process exits.
    WaitForSingleObject( pi.hProcess, INFINITE );

    // Close process and thread handles.
    CloseHandle( pi.hProcess );
    CloseHandle( pi.hThread );
}
```
