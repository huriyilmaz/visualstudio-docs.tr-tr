# <a name="c28284"></a>C28284
Uyarı C28284: Sembol için, koşullar işlev olmayan semboller için şu anda desteklenmiyor

 Bu uyarı, bir koşullu ek açıklamanın (koşul, `__drv_when`) işlev dışında bir öğe üzerinde bulunduğunu gösterir.
