# <a name="c28217"></a>C28217
Uyarı C28217: işlev Için, ek açıklamanın parametre sayısı dosyada bulunan ile eşleşmiyor

 Geçerli satırdaki ve iletideki satırdaki ek açıklamalar tutarsız. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının normal kullanımda görülebilmesi beklenmez.
