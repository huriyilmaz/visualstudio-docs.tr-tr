# <a name="c28225"></a>C28225
Uyarı C28225: ek açıklamada gerekli parametrelerin doğru sayısı bulunamadı

 Bu uyarı, adlandırılmış ek açıklamanın yanlış parametre sayısıyla kullanıldığını gösterir. Bu, standart makrolar ek açıklamalar için kullanılıyorsa mümkün olmamalıdır; Bu uyarının tipik kullanımda görülebilmesi beklenmez.
