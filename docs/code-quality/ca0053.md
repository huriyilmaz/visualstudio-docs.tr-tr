# <a name="ca0053"></a>CA0053
Kural derlemesi yüklenemedi.

Bu, bir kural derlemesi yansıma tarafından yüklenemedi gösterir. Kullanıcının doğru şekilde kurulmamıştır özel kuralları varsa veya analiz aracı derlemeyi dosya sistemi erişimi reddedildi gibi başka nedenlerle yükleyenene kadar bu durum oluşabilir.
