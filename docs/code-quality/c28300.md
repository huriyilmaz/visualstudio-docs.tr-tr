# <a name="c28300"></a>C28300
Uyarı C28300: < parameter_name >: İşleç < operator_name > için uyumsuz türlerin ifade işlenenleri

 Bu uyarı, bir parametresindeki işlenen türleri işleçle uyumlu olmadığında raporlanır.
