# <a name="list-of-available-services"></a>Kullanılabilir Hizmetlerin Listesi

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]ve Visual Studio SDK aşağıdaki hizmetleri destekler. Bazı paketler burada listelenmiyor kendi hizmetlerini sunar; örneğin, dil hizmetlerinin tek bir hizmet GUID'i yok.
