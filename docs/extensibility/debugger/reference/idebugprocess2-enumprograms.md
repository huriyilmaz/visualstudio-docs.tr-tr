# <a name="idebugprocess2enumprograms"></a>IDebugProcess2::EnumPrograms
Bu işlemde yer alan tüm programların listesini alın.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT EnumPrograms( 
   IEnumDebugPrograms2** ppEnum
);
```

```csharp
int EnumPrograms( 
   out IEnumDebugPrograms2 ppEnum
);
```

## <a name="parameters"></a>Parametreler
`ppEnum`\
[out] İşlemde yer alan tüm programların listesini içeren bir [IEnumDebugPrograms2](../../../extensibility/debugger/reference/ienumdebugprograms2.md) nesnesi döndürür.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa `S_OK` döndürür; aksi takdirde bir hata kodu döndürür.
