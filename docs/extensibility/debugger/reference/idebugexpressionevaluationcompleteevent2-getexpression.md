# <a name="idebugexpressionevaluationcompleteevent2getexpression"></a>IDebugExpressionEvaluationCompleteEvent2::GetExpression
Özgün ifadeyi alır.

## <a name="syntax"></a>Sözdizimi

```cpp
HRESULT GetExpression( 
   IDebugExpression2** ppExpr
);
```

```csharp
int GetExpression( 
   out IDebugExpression2 ppExpr
);
```

## <a name="parameters"></a>Parametreler
`ppExpr`\
[out] Ayrıştırılmış ifadeyi temsil eden bir [IDebugExpression2](../../../extensibility/debugger/reference/idebugexpression2.md) nesnesi döndürür.

## <a name="return-value"></a>Dönüş Değeri
 Başarılı olursa `S_OK` döndürür; aksi takdirde bir hata kodu döndürür.

## <a name="remarks"></a>Açıklamalar
 Bu yöntem, [ParseText](../../../extensibility/debugger/reference/idebugexpressioncontext2-parsetext.md) yöntemine yapılan bir çağrıda oluşturulan nesneyi döndürür.
