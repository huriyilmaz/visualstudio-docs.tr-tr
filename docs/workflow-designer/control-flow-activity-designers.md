# <a name="control-flow-activity-designers"></a>Akış Etkinliği Tasarımcılarını denetleme

Bu bölüm, bir iş akışı içindeki akışı denetlemek için kullanılan sistem tarafından sağlanmış etkinlikleri içerir.
