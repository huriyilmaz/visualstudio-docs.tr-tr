# <a name="flowchart-activity-designers"></a>Flowchart Etkinlik Tasarımcıları

Bu bölüm, bir iş akışı içindeki akış çizelgeleriyle yönetmek için kullanılan sistem tarafından sağlanmış etkinlikleri içerir.
