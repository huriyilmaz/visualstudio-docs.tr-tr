# <a name="environment-options-dialog-box"></a>Ortam Seçenekleri İletişim Kutusu
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Tümleşik geliştirme ortamının (IDE) belirli öğelerinin nasıl görüntüleneceğini ve davranacağını belirlemek için **Seçenekler** Iletişim kutusundaki **ortam** klasöründeki sayfaları kullanabilirsiniz. Ortam sayfalarına, menü çubuğunda **Araçlar**, **Seçenekler**ve ardından **ortam**' ı seçerek erişebilirsiniz.

> [!NOTE]
> Etkin ayarlarınıza ve kullandığınız Visual Studio sürümüne bağlı olarak, bilgisayarınızda görünen iletişim kutuları ve menü komutları Yardım'da açıklananlardan farklı olabilir. Ayarlarınızı değiştirmek için **Araçlar** menüsünden **Içeri ve dışarı aktarma ayarları** ' nı seçin. Daha fazla bilgi için bkz. [Visual Studio 'Da geliştirme ayarlarını özelleştirme](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).
