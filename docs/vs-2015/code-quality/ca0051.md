# <a name="ca0051"></a>CA0051

Hiçbir kural seçilmedi.

Bu hata, yönetilen kod analizi aracının etkinleştirildiğini, ancak hiçbir kuralın seçilmediğini gösterir. Bu hatayı düzeltmek için uygun kurallar ' ı seçin.
