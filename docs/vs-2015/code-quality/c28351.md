# <a name="c28351"></a>C28351
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Uyarı C28351: ek açıklama \<ek açıklaması >, koşulda bir dinamik değerin (bir değişken) kullanılamayacağını açıklar.  
  
 Bu uyarı, C/C++ türünün incelenebildiği bir ek açıklama uygulandığında raporlanır.  
  
 `_When_` ifade bir sabit değer vermelidir. `_When_` yok sayılır.
