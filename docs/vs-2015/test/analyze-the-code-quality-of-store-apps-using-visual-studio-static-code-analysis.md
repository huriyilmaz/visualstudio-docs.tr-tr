# <a name="analyze-the-code-quality-of-store-apps-using-visual-studio-static-code-analysis"></a>Visual Studio statik Kod analizini kullanarak Mağaza uygulamalarının kod kalitesini çözümleyin
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Visual Studio 'daki Kod Analizi Aracı, bir dizi yaygın kusur ve iyi programlama uygulaması ihlalleri için kodunuzu inceler. Kod Analizi uyarıları derleyici hatalarından ve uyarılarından farklıdır, ancak siz veya kodunuzu kullanan diğer kişiler için hala sorun oluşturabilir. Kod Analizi, kodunuzda test aracılığıyla keşfedilmesi zor olan kusurları da bulabilir. Geliştirme işleminiz sırasında kod analizi aracını düzenli aralıklarla çalıştırmak, tamamlanmış uygulamanızın kalitesini artırabilir.

> [!NOTE]
> Kod Analizi JavaScript kodu için kullanılamaz.
