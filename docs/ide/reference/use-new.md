# <a name="use-new"></a>`new()` komutunu kullanma

Bu için geçerlidir:

- C#

**Ne:** Kullanın `new()` .

**Ne zaman:** Kullanmayan bir alanınız `var` veya bir kod stili tercihi kullanılamaz `var` .

**Neden:** Bu nedenle, türü iki kez tekrarlayarak yinelenen kod yazmanız gerekmez.
