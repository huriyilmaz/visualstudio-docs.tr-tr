# <a name="-devenvexe"></a>/? (devenv.exe)

Her bir anahtarın kısa bir açıklamasıyla birlikte tüm `devenv` anahtarlarını listeleyerek bir ileti kutusu görüntüler.

## <a name="syntax"></a>Sözdizimi

```shell
devenv /?
```
