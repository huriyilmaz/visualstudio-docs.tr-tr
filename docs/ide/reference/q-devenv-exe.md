# <a name="-devenvexe"></a>/? (devenv.exe)

Tüm anahtarları listeleen `devenv` ve her anahtarın kısa bir açıklamasının yer alan bir ileti kutusu görüntüler.

## <a name="syntax"></a>Syntax

```shell
devenv /?
```
