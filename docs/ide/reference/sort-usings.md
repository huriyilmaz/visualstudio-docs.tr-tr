Bu yeniden düzenleme için geçerlidir:

- C#

- Visual Basic

**Ne:** Using deyimlerini sıralayın.

**Ne zaman:** `using` Dosyanızın en üstündeki yönergeleri alfabetik sırada olacak şekilde sıralamak istiyorsunuz.
