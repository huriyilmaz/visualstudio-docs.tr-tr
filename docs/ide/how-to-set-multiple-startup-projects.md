# <a name="how-to-set-multiple-startup-projects"></a>Nasıl yapılır: birden çok başlangıç projesi ayarlama

Visual Studio, hata ayıklayıcıyı başlattığınızda birden çok projenin nasıl çalıştırılacağını belirtmenizi sağlar.

## <a name="to-set-multiple-startup-projects"></a>Birden çok başlangıç projesi ayarlamak için

1. **Çözüm Gezgini**, çözümü (en üstteki düğüm) seçin.

2. Çözüm düğümünün bağlamını (sağ tıklama) menüsünü ve ardından **Özellikler**' i seçin. **Çözüm Özellik sayfaları** iletişim kutusu görüntülenir.

3. **Ortak özellikler** düğümünü genişletin ve **Başlangıç projesi**' ni seçin.

4. **Birden çok başlangıç projesi** seçeneğini belirleyin ve uygun eylemleri ayarlayın.
